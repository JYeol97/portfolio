```markdown
## 🛠️ 기술 스택 (Tech Stack)

### 📦 프로젝트 빌드 & 구성

| 도구 / 프레임워크        | 설명 |
|-------------------------|------|
| **Vite**                | 초고속 프론트엔드 빌드 도구로, React 개발 환경을 빠르게 구성하기 위해 사용함 |
| **pnpm**                | 빠르고 디스크 효율적인 패키지 매니저. 종속성 관리와 워크스페이스 구성에 최적화됨 |
| **TypeScript**          | 정적 타입을 통해 코드 안정성과 유지보수성을 높이기 위해 사용함 |
| **ESLint**              | 코드 품질 및 스타일 일관성을 유지하기 위한 정적 분석 도구 |
| **Prettier**            | 코드 포맷 자동화 도구 (선택사항 – 사용 중이라면 포함) |

---

### 🎨 UI & 스타일링

| 도구 / 라이브러리                | 설명 |
|------------------------------   |------|
| **React**                       | 사용자 인터페이스 구축을 위한 UI 라이브러리 |
| **Tailwind CSS**                | 유틸리티 기반 CSS 프레임워크로, 빠르고 일관된 스타일링 가능 |
| **shadcn/ui**                   | Tailwind 기반의 접근성 높은 UI 컴포넌트 모음 |
| **class-variance-authority**    | 컴포넌트 스타일을 variant 기반으로 동적 제어하기 위해 사용 |
| **tailwindcss-animate**         | Tailwind에서 애니메이션 유틸리티를 확장하기 위한 플러그인 |
| **Framer Motion**               | 리액트용 애니메이션 라이브러리. 스크롤 기반 섹션 전환, 페이드 인/아웃 등 구현 가능 |
| **react-intersection-observer** | 컴포넌트가 화면에 들어왔는지 감지하는 Hook 제공 (Framer Motion과 잘 결합됨) |

---

### 🧱 컴포넌트 & 접근성

| 도구 / 라이브러리             | 설명 |
|------------------------------|------|
| **@radix-ui/react-slot**     | `shadcn/ui` 내부에서 Slot을 사용하여 유연한 컴포넌트 구성 지원 |
| **@types/node**              | Node.js 관련 타입 정의. Vite 설정이나 `vite.config.ts`에서 `path` 사용을 위해 필요 |
```
